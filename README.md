# 🐾 Cat: Midnight Mischief

Welcome to **Cat: Midnight Mischief**! This is a fun text-based adventure game written in Python where you play as a mischievous cat navigating the house at night. 

Will you successfully knock over the water glass, or will you get caught by the humans?

---

## 🎮 How to Play

1. Make sure you have Python installed on your computer.
2. Download or copy the `index.py` file to your computer.
3. Open the file using a Python editor (like IDLE) and press **Run**.
4. Read the story prompts on the screen and type in your choices to guide your cat through the night!

---

## 🚀 Features
* Fun text-based choices and multiple storylines.
* Authentic cat behavior (mischief, naps, and snacks).
* Multiple different endings based on the choices you make.
